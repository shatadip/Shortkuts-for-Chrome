Shortkuts

  1. Use right-click or hold the Alt key and scroll to switch tabs ✓
  2. Right-click / Alt + Mouse Wheel Up: Switch to the previous tab ✓
  3. Right-click / Alt + Mouse Wheel Down: Switch to the next tab ✓
  4. Alt + Shift + →: Close all tabs to the right ✓
  5. Alt + Shift + ←: Close all tabs to the left ✓
  6. Alt + Shift + ↑: Close all other tabs ✓
  7. Alt + Shift + D: Duplicate Tab ✓
  8. Alt + P: Toggle Play/Pause X

Chrome Default Shortcuts

  1. Alt + D: Focus on the address bar
  2. Ctrl+ Shift + B: Show/hide bookmarks bar
  3. Alt + F: Open the Chrome menu
  4. Ctrl + N: Open a new Chrome window
  5. Ctrl + Shift + N: Open a new incognito window
  6. Ctrl + 1 to Alt + 8: Switch to the corresponding tab in the tab strip
